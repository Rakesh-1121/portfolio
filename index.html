ConnectGen: Support System

Project Overview

ConnectGen is a full-stack customer support and FAQ management system designed to streamline user inquiries and admin responses. The platform bridges users and support teams with an intuitive interface for submitting, tracking, and resolving support requests.

Key Features
  • User Portal: Submit support requests, view request history, edit pending requests, and track status updates (Pending, In Progress, Resolved)
  • Admin Dashboard: Comprehensive dashboard with real-time statistics, filtering, sorting, and response management capabilities
  • FAQ System: Searchable FAQ database with accordion-style interface for quick answers
  • Status Tracking: Color-coded status badges and request lifecycle management
  • Real-time Updates: Character counter, form validation, and flash message notifications
Technical Implementation

Built using Flask (Python) for the backend with session management and database integration. The frontend leverages Bootstrap 5 for responsive design, JavaScript for dynamic filtering/sorting, and Font Awesome for iconography. The system includes 6+ interconnected pages with full CRUD functionality.

Technologies:

Python Flask SQLite Bootstrap 5 JavaScript HTML/CSS
